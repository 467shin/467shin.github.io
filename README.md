   > 글자 크기가 대충 화면 크기에 맞춰진 다는 것은 알고 있었지만 브라우저의 기본 폰트 사이즈를 기준으로 한다는 사실은 처음 알았다.
2. `border-radius`의 값은 반지름의 길이
   > %는 `width`값을 기준으로 하는 퍼센티지이다.
